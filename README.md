# Disable Ctrl+Q
WebExtension that disables Ctrl+Q in Firefox  
    
Only works on Windows and MacOS. Doesn't work on Linux until this bug is addressed.  
https://bugzilla.mozilla.org/show_bug.cgi?id=1325692
